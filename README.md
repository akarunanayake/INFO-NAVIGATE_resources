This repository contains the resources of tool INFO-NAVIGATE, which is designed as a part of socio-technical pedagogical intervention to support students in identifying high-quality information in the context of conflicting information. INFO-NAVIGATE is a LLMs based web tool. It accepts a search query as the user input and generates the content utilizing the NLP capabilities of LLMs.
