This repository contains the resources of tool INFO-NAVIGATE, which is designed as a part of socio-technical pedagogical intervention to support higher edication students in identifying high-quality information in the context of conflicting information to make informed decisions. INFO-NAVIGATE is a GenAI based web tool. It accepts a search query as the user input and generates the content utilising the NLP capabilities of LLMs. 

Tech Stack:
1. Backend - Python (Flask)
2. LLM API - OpenAI API
3. Frontend - JavaScript, Ajax
4. Databse - MySQL
5. Deployment - CI/CD pipeline, AWS
